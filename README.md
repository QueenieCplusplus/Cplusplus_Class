# Limitation of Overload 

多載的運算子不一定是成員函數，但是至少一運算元必須是使用者自訂型態。

1. 不產生新的運算子符號。

2. 無法多載特定的運算子，例如 ?: (條件運算子)、. (成員運算子)、.* (成員指標運算子)、sizeof、cast。
